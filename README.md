Project Title:
Online Retail Sales Database Design

Project Description
A comprehensive MySQL database solution designed for managing complete online retail operations. This project features a fully normalized 8-table schema with real sample data, analytical queries, and reporting views for e-commerce platforms. It demonstrates professional database design principles including entity relationships, data integrity constraints, and business intelligence capabilities.

Normalization - 3NF

Entities:
. Customers
. Categories
. Products
. Inventory
. Orders
. Order_Items
. Payments
. Shipping

Tools Used:
MYSQL Workbench
